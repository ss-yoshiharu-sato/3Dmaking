# Summary

Hi,

a friend of mine made this universal filament holder. You can mount filament spools in a wide range. For maximum rated dimensions see attached picture. I tried up to 2.6kg material spools without an issue. For smooth unwinding it uses ball bearings. 

What you need:
- M8 threaded rod
- 2x M8 nut
- 2x skate board ball bearing 608ZZ
  - 8mm (core)
  - 22mm (outer diameter)
  - 7mm (width)
